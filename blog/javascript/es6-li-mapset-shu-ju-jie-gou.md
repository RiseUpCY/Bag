# ES6里MapSet数据结构

> Map是一种类似于json的数据格式，map的键key可以是任意类型\*/
>
> ```text
> let m = new Map()
> // 添加
> m.set(123, 'abc').set(234, 'bcd').set(456, 'doh')
> // 获取
> m.get(123) // 'abc'
> // 删除
> m.delete(456)
> // 判断是否有key
> m.has(123)  //true
> // 清空
> // m.clear()
> // 遍历
> for(let [key, val] of m.entries()){
>     console.log(`${key} -- ${val}`)
> }
> // 遍历
> m.forEach((val, key) => {
>     console.log(key, val)
> })
> ```
>
> WeakMap
>
> ```text
> {//weakmap的key只能是对象
>     let m = new WeakMap();
>     let json = {a: 1, b: 2};
>     m.set(json,1);
>     console.log(m)
> }
> ```
>
> Set// 类似数组的数据结构 set的值不能重复
>
> ```text
> // set里放的是一个 数组
> let s = new Set(['a', 'b', 'c', 'd'])
> // 添加一个值
> s.add('e')
> // 删除
> s.delete('b')
> // 检测是否有
> s.has('a') //true
> // 查看值
> s.values()
> // 长度
> s.size
> // 清空
> // s.clear()
> for(let val of s){
>     console.log(`${val}`)
> }
> ```
>
> set特性可以用于数组的去重
>
> ```text
> let arr = [1,3,2,6,5,3,2,16,5,2,3,1,6,2,3,1];
> let newarr = [...new Set(arr)]
> ```
